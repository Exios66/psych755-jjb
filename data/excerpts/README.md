# Excerpt fixtures

Canonical filenames consumed by `config/default.yaml` and the analysis package:

- `prolific_excerpt.csv` — Prolific demographics export
- `qualtrics_excerpt.csv` — Qualtrics survey export (3-row header)

Original export filenames (same content) are also kept for reference:

- `Excerpt For Testing Prolific Export.csv`
- `Excerpt For Testing Qualtrics Export.csv`

Join key: Prolific `Participant id` ↔ Qualtrics `PROLIFIC_PID` / `Q0`.
